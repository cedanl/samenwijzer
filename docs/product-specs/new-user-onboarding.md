# New User Onboarding

**Status:** draft

## Goal

A student opens the app for the first time and understands within 60 seconds
what Samenwijzer does and how to get started.

## Acceptance criteria

- [ ] Welcome screen explains the purpose in plain language.
- [ ] Student can start a learning session without creating an account.
- [ ] First AI interaction happens within 2 clicks from landing.

## Out of scope

- Account creation / authentication (future milestone).
