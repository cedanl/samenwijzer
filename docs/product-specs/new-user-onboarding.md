# New User Onboarding

**Status:** done (fase 1)

## Goal

Een student opent de app voor de eerste keer en begrijpt binnen 60 seconden
wat Samenwijzer doet en hoe te beginnen.

## Acceptance criteria

- [x] Welcome screen explains the purpose in plain language.
- [x] Student can start a learning session without creating an account.
- [x] First AI interaction happens within 2 clicks from landing.

## Flow (fase 1)

1. **Landing** (`/`): student leest uitleg, kiest naam uit lijst.
2. **Klik op "Ga naar AI Tutor →"**: chatpagina opent met begroeting.
3. Student typt eerste bericht → tutor antwoordt.

Totaal: 2 klikken (naam kiezen telt als 1 actie, navigatielink als 2e).

## Fase 2

- Authenticatie met studentnummer of SSO (als gewenst).
- Onboardingflow met korte intro-animatie of rondleiding.
