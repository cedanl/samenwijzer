## What this project is

A Python application using AI and Data to support students in learning.
Target users: students (external alpha testers) and internal daily users.

## Core rules

1. **No handwritten code.** Every line of code is generated by Claude Code.
2. **No business logic in the app layer.** `app/main.py` is UI only; logic lives in `src/samenwijzer/`.
3. **Dependency direction:** `prepare → transform → analyze → visualize → export`. Never reversed.
4. **Structured logging only.** No bare `print()` calls in production code.
5. **Validate at boundaries.** Validate external data on ingestion; trust internal types.
